## ⚡ Running a Project


## 🛠️ Setup

```bash
git clone https://github.com/kiwisigma/100-days-of-python.git
cd 100-days-of-python
python --version  # requires Python 3.10+
```

Each day folder is self-contained — no shared dependencies unless noted.
