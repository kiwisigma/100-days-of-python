# Day 7 — Hangman Project

## What I Built
A fully playable Hangman game — the first complete multi-file project of the course.

## Features
- Random word selection
- Letter-by-letter guessing with lives tracking
- ASCII art gallows that updates each wrong guess
- Duplicate guess detection

## Concepts Practiced
- Functions with parameters and return values
- List and string manipulation
- Game loop logic
- Importing from local modules

## How to Run
```bash
python Hangman.py
```
