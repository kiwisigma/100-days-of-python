# Day 4 — Randomisation & Lists

## What I Built
- **Heads or Tails** — coin flip simulator using the `random` module
- **Rock Paper Scissors** — classic game against the computer
- **Who Pays the Bill** — randomly picks who pays from a list of names

## Concepts Practiced
- `random.randint()`, `random.choice()`
- Python lists: indexing, appending, nested lists
- `import` and using modules

## How to Run
```bash
python head_or_tail.py
python rock_paper_scissors.py
python "who pays the bill.py"
```
